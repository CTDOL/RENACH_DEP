# RENACH_DEP

Este projeto consiste em um aplicativo desenvolvido em **APEX** (Oracle Application Express). 

## Objetivo
O objetivo principal deste APP é gerar uma tabela no formato **CSV** contendo a relação dos **APROVADOS** nos cursos **CVE** (Curso de Veículos de Emergência) e **ACVE** (Atualização do Curso de Veículos de Emergência), a fim de enviar essas informações para o **SENATRAN**.
